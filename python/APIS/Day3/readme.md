1. Password Attempts
A user has 3 attempts to enter the correct password.
•	Ask the user for the password.
•	If it is correct, display "Access Granted".
•	If incorrect, reduce the number of attempts.
•	After 3 failed attempts, display "Account Locked".
 
2. Bank Account
A user starts with £500.
•	Ask how much they want to withdraw.
•	Do not allow them to withdraw more than their balance.
•	Update the balance.
•	Continue until they enter 0.
 
3. Quiz Game
Create a quiz with 5 questions.
•	Ask each question.
•	Give 1 mark for each correct answer.
•	Display the final score.
•	Tell the user if they passed (4 or more).
 
4. Temperature Logger
Ask the user to enter 7 daily temperatures.
•	Keep track of:
o	highest temperature
o	lowest temperature
o	average temperature
•	Display all three at the end.
 
5. Shopping Basket
Items cost £4 each.
•	Ask how many items the customer wants.
•	Calculate the total cost.
•	If the total is over £30, apply a 15% discount.
•	Display the final price.
 
6. Exam Marks
Ask the user to enter marks for 10 students.
•	Count how many passed (50 or above).
•	Count how many failed.
•	Display the highest mark.
•	Display the average mark.
 
7. PIN Machine
The correct PIN is 4826.
•	Give the user 3 attempts.
•	If correct, display "Welcome".
•	Otherwise display "Incorrect PIN".
•	Lock the account after 3 failures.
